# Activity History  

The Activity History tab is a dedicated section to help track the progress of each proposal.

This tab displays key actions with its corresponding dates.


Specifically the Activity History shows when:
- Funds Received
- Extension Granted
- Duration Ended
- Report Submitted
- Proposal Delivered
- Proposal Flagged
- Proposal Overdue
